== Description ==

Crunchy Sharing is a fastest, optimized, NO-Javascript based, light weight, GDPR compliant Social sharing buttons for your site. Plugin provides more 25 different configuration options for your site.

Crunchy Sharing is created with performance in mind. Without making any query to Database, External API endpoint, Crunchy Social Sharing button loads in fraction of seconds.

**Key features includes**

* Post/Page and any Custom Post Type (CPT) based social sharing button integration
* Floating Social Sharing button
* Mobile sticky bottom social sharing options
* [crunchy_social_sharing] short code integration
* Auto display to Post, Page, Media and Custom Post type
* 100% responsive
* Reorder Social icon order easily
* Before content, After content or Adding top/bottom both options
* Add text and align your custom text before social share icons

== Installation ==

= Installing the plugin =

1. Unpack the crunchy-sharing.zip.
2. Upload the file to the /wp-content/plugins/ directory.
3. Activate the plugin through the Plugins menu in WordPress.
4. Configure the options under Admin Panel Crunchy Sharing -> General Options.
5. Activate your license by putting your activation key which you got in your email under Admin Panel Crunchy Sharing -> Activate License.
6. Done and Ready.
